🛡️ About Sentinel-X

Sentinel-X is a Next-Generation Firewall (NGFW) powered by Artificial Intelligence. Unlike traditional firewalls that follow strict rules, Sentinel-X learns "normal" network behavior and actively blocks anomalies using a Random Forest Machine Learning model.

Cyber Threat Glossary

The "Half-Open" Attack.
Imagine a prankster calling a pizza shop, ordering a pizza, and hanging up before giving the address. The shop waits (using resources) but never finishes the order. In a SYN Flood, the attacker sends thousands of "Hello" (SYN) packets but never completes the handshake (ACK), exhausting the server's memory until it crashes.

Knocking on Every Door.
Before robbing a house, a thief checks every window to see which one is unlocked. Similarly, hackers scan your network ports (like 80, 443, 22) to find vulnerable services. Sentinel-X detects this rapid sequential access and blocks the scanner immediately.

The Hidden Bomb.
Normal web requests are small. If a packet suddenly contains 2000+ bytes of data when it should be 50 bytes, it might contain a "Buffer Overflow" exploit code. Our AI analyzes packet metadata (size, flags, timing) to spot these subtle deviations.
